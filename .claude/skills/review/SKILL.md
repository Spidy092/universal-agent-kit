---

Read `AGENTS.md` first, then use `.agents/commands/review.md` and `.agents/skills/code-review/SKILL.md`.

Workflow:
- Review requested files, branch, PR, or diff.
- Lead with findings ordered by severity.
- Include file and line references when possible.
- Do not edit unless explicitly asked.
- End with missing tests and merge readiness.
