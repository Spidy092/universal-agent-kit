# RTK Prompt Template

```txt
Role:
You are a senior [role].

Task:
[Exact task]

Knowledge:
[Project context, files, errors, constraints]

Rules:
- Read existing code first.
- Do not assume.
- Make minimal safe changes.
- Give exact files.
- Give exact commands.
- Mention edge cases.

Output:
- Summary
- Root cause / plan
- Files to change
- Code/fix
- Commands
- Verification
- Edge cases
```
