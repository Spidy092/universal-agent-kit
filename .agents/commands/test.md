# Command: /test

Uses:
- `testing`
- Add domain skills when tests expose a bug or implementation gap.

Examples:

```txt
/test
```

```txt
/test add regression coverage for user registration
```

Workflow:
1. Identify the behavior and risk to cover.
2. Use existing test patterns.
3. Add focused tests before broad rewrites.
4. Run the narrowest useful test command first.
