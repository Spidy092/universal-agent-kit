# Command: /docs

Aliases:
- `/docs`
- `/research`

Uses:
- `docs-research`

Examples:

```txt
/docs check latest OpenWA docs
```

```txt
/research verify current Stripe webhook signature API
```

Workflow:
1. Prefer official documentation and primary sources.
2. Verify current API, CLI, or library behavior before changing code.
3. Cite or summarize the source used.
4. Apply only the relevant finding.
