# /help

Aliases: none

Skills: none by itself.

Use when the user wants available commands, modes, examples, or routing help.

Required behavior:
- Read `AGENTS.md` first.
- Summarize available commands from `.agents/commands`.
- Summarize available modes from `.agents/modes`.
- Give concise examples.
- Mention that slash commands work as in-agent workflow instructions even when the host agent has no native slash-command support.

Output format:
- Commands
- Modes
- Examples
